# 附录

附录用于辅助检索和统一术语。正文按历史章节展开，附录按“人名、术语、时间、书目”四类组织，适合写作、复习和交叉查找时使用。

## 使用方式

- [人名索引](人名索引.md)：按人物列出所属传统、代表概念和主要阅读章节。
- [术语词典](术语词典.md)：统一中英术语对照，避免同一概念在不同章节中译名混乱。
- [时间线](时间线.md)：按事件和著作把思想史顺序串联起来。
- [参考书目](参考书目.md)：列出正文使用的经典文本和通行延伸阅读。

附录不是正文替代品。人名索引帮助定位人物，术语词典帮助理解关键词，时间线帮助建立先后关系，参考书目帮助继续阅读。研究写作时，应回到正文中的论证结构，而不是只引用附录中的简短定义。
