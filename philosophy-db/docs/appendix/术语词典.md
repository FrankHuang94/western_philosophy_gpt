# 术语词典

本词典统一正文中的中英术语。定义为简明工作定义，详细论证见对应章节。

| 中文术语 | 英文术语 | 简明定义 | 相关章节 |
| --- | --- | --- | --- |
| 理念论 | Theory of Forms | 柏拉图关于可感事物依赖超感性理念的理论。 | 01 |
| 四因说 | Four Causes | 亚里士多德用质料因、形式因、动力因、目的因解释事物。 | 01 |
| 唯名论 | Nominalism | 否认普遍概念作为独立实体存在的立场。 | 02 |
| 方法论怀疑 | Methodological Doubt | 笛卡尔为寻找确定性而暂时怀疑一切可疑信念的方法。 | 03 |
| 心物二元论 | Mind-body Dualism | 认为心灵和物体属于不同实体或属性领域。 | 03 |
| 实体一元论 | Substance Monism | 斯宾诺莎认为只有一个无限实体的本体论。 | 03 |
| 白板说 | Tabula Rasa | 洛克认为心灵没有天赋观念，知识来自经验。 | 03 |
| 休谟之叉 | Hume's Fork | 将知识区分为观念关系和事实问题。 | 03 |
| 公意 | General Will | 卢梭关于共同体公共意志和政治合法性的概念。 | 04 |
| 公开运用理性 | Public Use of Reason | 康德关于公共讨论和启蒙的核心概念。 | 04 |
| 先验综合判断 | Synthetic A Priori Judgment | 康德认为既扩展知识又具有先天必然性的判断。 | 05 |
| 物自体 | Thing-in-itself | 康德用于标示经验对象之外不可被理论认识穷尽之物。 | 05 |
| 辩证法 | Dialectic | 通过矛盾、否定和发展理解概念或历史运动的方法。 | 05 |
| 扬弃 | Sublation | 黑格尔概念，指否定、保存和提升的统一过程。 | 05 |
| 意志 | Will | 叔本华用于说明世界深层盲目欲求的概念。 | 06 |
| 主体性真理 | Subjective Truth | 克尔凯郭尔关于真理必须由个人存在性承担的思想。 | 06 |
| 透视主义 | Perspectivism | 尼采认为认识总从特定生命立场和解释视角出发。 | 06 |
| 权力意志 | Will to Power | 尼采关于生命解释、组织和超越自身力量的概念。 | 06 |
| 实证主义 | Positivism | 主张知识应限于经验事实和科学规律的立场。 | 07 |
| 功利主义 | Utilitarianism | 以幸福、快乐或效用最大化评价行为和制度的伦理理论。 | 07 |
| 历史唯物主义 | Historical Materialism | 马克思关于生产方式和社会关系解释历史变迁的理论。 | 07 |
| 异化 | Alienation | 人的活动、产品、社会关系反过来支配人的状态。 | 07 |
| 意向性 | Intentionality | 意识总是关于某物、指向某物的结构。 | 08 |
| 现象学还原 | Phenomenological Reduction | 悬置自然态度，分析对象如何在经验中显现的方法。 | 08 |
| 此在 | Dasein | 海德格尔对能理解自身存在的人类存在方式的称呼。 | 08 |
| 在世存在 | Being-in-the-world | 海德格尔关于人总已处在实践世界中的存在结构。 | 08 |
| 存在先于本质 | Existence Precedes Essence | 萨特关于人先存在并通过行动形成自身的命题。 | 08 |
| 他者 | Other | 波伏娃用于分析女性被设定为男性主体之外部位置的概念。 | 08 |
| 荒诞 | Absurd | 加缪指人的意义要求与世界沉默之间的张力。 | 08 |
| 视域融合 | Fusion of Horizons | 伽达默尔关于解释者视域与文本视域相互改变的概念。 | 08 |
| 意义与指称 | Sense and Reference | 弗雷格区分对象被给予方式和所指对象。 | 09 |
| 摹状词理论 | Theory of Descriptions | 罗素用逻辑分析处理描述短语和存在性难题。 | 09 |
| 可证实性原则 | Verification Principle | 逻辑实证主义用经验验证区分认知意义的原则。 | 09 |
| 语言游戏 | Language Game | 后期维特根斯坦关于意义在使用规则中形成的概念。 | 09 |
| 言语行为 | Speech Act | 说话本身可构成承诺、命令、命名等行动的理论。 | 09 |
| 可证伪性 | Falsifiability | 波普尔用于区分科学理论是否承担反例风险的标准。 | 09 |
| 范式 | Paradigm | 库恩关于科学共同体共享范例、方法和标准的概念。 | 09 |
| 刚性指称词 | Rigid Designator | 克里普克指在所有可能世界指称同一对象的名称。 | 09 |
| 实用主义准则 | Pragmatic Maxim | 皮尔士要求通过实践后果澄清概念意义的准则。 | 10 |
| 工具理性 | Instrumental Reason | 只计算达成目的的有效手段而不反思目的正当性的理性形式。 | 11 |
| 文化工业 | Culture Industry | 阿多诺关于资本主义大众文化标准化和整合功能的概念。 | 11 |
| 否定辩证法 | Negative Dialectics | 阿多诺强调非同一性并拒绝总体和解的辩证法。 | 11 |
| 交往理性 | Communicative Reason | 哈贝马斯关于以理解为目标的沟通理性。 | 11 |
| 能指/所指 | Signifier/Signified | 索绪尔语言符号的形式面和概念面。 | 12 |
| 符号任意性 | Arbitrariness of the Sign | 能指与所指之间没有自然必然联系。 | 12 |
| 话语 | Discourse | 福柯指生产对象、主体和真理规则的实践系统。 | 12 |
| 解构 | Deconstruction | 德里达通过细读揭示文本二元等级和中心不稳定的方法。 | 12 |
| 延异 | Differance | 德里达关于意义由差异和延迟生成的概念。 | 12 |
| 元叙事 | Metanarrative | 利奥塔指为知识和历史提供总体合法性的宏大故事。 | 13 |
| 拟像 | Simulacrum | 鲍德里亚指不再依赖原本的符号复制。 | 13 |
| 超真实 | Hyperreality | 符号系统制造出的比现实更可消费的现实感。 | 13 |
| 原初状态 | Original Position | 罗尔斯用于选择公平正义原则的思想实验。 | 14 |
| 无知之幕 | Veil of Ignorance | 原初状态中遮蔽个人社会位置和天赋的信息条件。 | 14 |
| 差异原则 | Difference Principle | 罗尔斯认为不平等只有在最有利于最不利者时才正当。 | 14 |
| 持有正义 | Entitlement Theory | 诺齐克关于取得、转让和纠正构成财产正义的理论。 | 14 |
| 无负荷自我 | Unencumbered Self | 社群主义批判自由主义预设脱离共同体的抽象主体。 | 14 |
| 性别表演 | Gender Performativity | 巴特勒关于性别由重复规范实践生成的理论。 | 14 |
