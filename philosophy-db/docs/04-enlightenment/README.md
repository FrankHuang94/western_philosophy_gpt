目标字数：25,000 中文字。核心重点章节。

## 待写作范围

### 法国启蒙运动

- 伏尔泰：宗教宽容、反教会专制、与卢梭的文明进步争论。
- 孟德斯鸠：政体分类、权力制衡、对洛克分权思想的发展。
- 狄德罗与《百科全书》：知识民主化与唯物主义倾向。
- 卢梭：公意、不平等起源、自然状态与文明批判。
- 孔狄亚克、爱尔维修：感觉论与唯物主义。

### 苏格兰启蒙运动

- 休谟道德情感论。
- 亚当·斯密：同情机制、市场理论与道德哲学关联。
- 弗格森：市民社会理论。

### 德国启蒙运动

- 莱辛：宗教宽容、戏剧与哲学论战。
- 门德尔松与康德：什么是启蒙的论战背景。
- 康德：敢于求知、公开运用理性与私下运用理性。

### 启蒙运动的内部张力

- 理性乐观主义与卢梭式文明批判。
- 苏格兰经验主义与法国唯理主义倾向。
- 启蒙理性与德国唯心主义的后续回应。
