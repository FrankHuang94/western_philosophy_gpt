- 弗雷格：意义与指称、对心理主义的批判。
- 罗素：逻辑原子主义、摹状词理论、罗素悖论。

- 维特根斯坦前期与后期：图像论到语言游戏。

### 科学哲学

- 波普尔、库恩、拉卡托斯、费耶阿本德。

- 奎因：对分析-综合区分的批判。
- 克里普克：命名与必然性、对描述主义指称理论的批判。
